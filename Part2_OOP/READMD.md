# Part 2 — Object-Oriented Programming

OOP fundamentals through progressively complex projects.

## Sessions

| Sessions | Topic | Project |
|----------|-------|---------|
| S16 – S17 | Classes & Instances | — |
| S18 – S19 | Inheritance | Shop Management CLI |
| S20 – S21 | Inheritance, Abstraction, Encapsulation, Polymorphism | — |
| S22 – S23 | Magic Methods, Custom Exceptions | User Profile System |
| S24 – S25 | Doctest & Custom Collections | Notes Manager |
| S26 – S29 | Abstract Classes, Properties, `getattr` | Hospital Management System |

## Projects

**Shop Management** (`S18_S19/Project/`)
> Manage inventory items with add, remove, edit, search, and stats.

**User Profile System** (`S22_S23/user_profile/`)
> Login/logout, email update, password change, and activity logging with custom exceptions.

**Notes Manager** (`S24_S25/Project/`)
> String-only custom list with filter, count, and unique operations. Includes doctest examples.

**Hospital Management System** (`S26_S27_28_29/hospital_project/`)
> Manages doctors, nurses, and patients via an abstract `Person` base class. Supports dynamic search with `**kwargs` and `getattr`.
