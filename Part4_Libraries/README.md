# Part 4 — Python Libraries

Data manipulation and visualization with NumPy, Pandas, and Matplotlib.

## Sessions

| Sessions | Topic | Files |
|----------|-------|-------|
| S38 – S39 | NumPy & Pandas basics | `Numpy2.ipynb`, `pandas2.ipynb` |
| S40 – S41 | Pandas advanced & Matplotlib | `pandas1.ipynb`, `plot.ipynb` |

## Topics Covered

**NumPy** — arrays vs lists, dtype, shape, indexing, slicing, filtering, vectorize, `copy` vs `view`

**Pandas** — Series, DataFrame, `loc`/`iloc`, CSV I/O, mapping

**Matplotlib** — line, scatter, pie, bar, histogram, box plot, subplots, 3D plots

## Datasets

| File | Description |
|------|-------------|
| `CoffeeAndCode.csv` | Survey data used in Pandas exercises |
| `healthcare-dataset-stroke-data.csv` | Medical dataset for real-world analysis |
